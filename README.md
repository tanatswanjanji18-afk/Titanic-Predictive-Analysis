# Titanic Survival Prediction Analysis

## Overview
This project analyzes the Titanic dataset to predict passenger survival using machine learning. We explore factors influencing survival and build predictive models with >80% accuracy.

**Project Business Questions:**
1. What demographic factors most affected survival chances?
2. Which machine learning model predicts survival most accurately?
3. What actionable insights can inform future safety protocols?

## Dataset
**Source:** [Kaggle Titanic Dataset](https://www.kaggle.com/c/titanic/data)  
**Size:** 891 passengers Ã— 12 features  
**Target Variable:** `survived` (0 = did not survive, 1 = survived)

**Key Features:**
- **Demographic:** sex, age, pclass
- **Family:** sibsp, parch
- **Ticket:** fare, embarked
- **Derived:** family_size, is_alone, age_group, fare_per_person
